# FinTrack - личный финансовый трекер

Сайт на React + TypeScript для контроля личных доходов и расходов с удобной помесячной аналитикой.

## Архитектура и структура компонентов

### Папка `src/Components/UI/`
Сюда вынесены компоненты, которые не привязаны к бизнес-логике финансов, полностью универсальны и могут быть переиспользованы в любом другом проекте:
*   `Button` - кастомная кнопка в фирменных нежно-розовых тонах.
*   `Input` - универсальное текстовое/числовое поле ввода.
*   `Radio` - гибкий переключатель для выбора типа транзакции (Расход/Доход).
*   `Select` - выпадающий список для выбора категорий трат или поступлений.

### Папка `src/Components/Widgets`
Крупные компоненты, которые управляют состоянием приложения и отвечают за функционал трекера:
*   `TransactionForm` — форма добавления новых финансовых операций с использованием базовых UI-компонентов.
*   `TransactionList` — интерактивная таблица для вывода истории всех совершённых операций.
*   `Analytics` — блок аналитических карточек, рассчитывающий чистый итог за выбранный период.
*   `MonthSelector` - панель навигации для быстрого переключения между месяцами 2026 года.

## Технологический стек
*   **Vite** — быстрый инструмент сборки.
*   **React** & **TypeScript** — для строго типизированного и компонентного UI.
*   **CSS Modules** — для изолированных и безопасных стилей компонентов.
