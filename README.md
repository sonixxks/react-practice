# CashGlow - Финансовый трекер

Это современное веб-приложение для контроля и анализа личных финансов.

## Ключевой функционал

* **Локальная авторизация и сессии:** Авторизации пользователей с генерацией и валидацией токенов авторизации на клиенте.
* **Изоляция данных:** Все транзакции привязаны к `username` авторизованного пользователя. Разные учетные записи на одном устройстве имеют изолированные персональные данные в `localStorage`.
* **Управление транзакциями:** Добавление доходов и расходов с указанием даты, категории и названия, а также автоматический расчет текущего баланса за выбранный месяц.
* **Интерактивная годовая аналитика:** Визуализация финансовой динамики по месяцам текущего года с помощью кастомизированной гистограммы на Chart.js.
* **Умный рейтинг трат:** Автоматическая группировка расходов за год и сортировка категорий от самых затратных к меньшим.

## Технологический стек

* **Ядро:** React (v18+), Vite
* **Язык программирования:** TypeScript
* **Управление состоянием:** React Context API
* **Маршрутизация:** React Router DOM (защищенные приватные роуты `ProtectedRoute`)
* **Стилизация:** SCSS Modules (изолированные модульные стили, адаптивная верстка)
* **Визуализация данных:** Chart.js
