# CashGlow — Личный финансовый трекер

**CashGlow** — это современное веб-приложение (SPA) для эффективного контроля, категоризации и визуального анализа личных финансов. Проект разработан с акцентом на безопасность локальных данных, строгую типизацию и компонентный подход.

## Ключевой функционал

* **Надежная валидация форм:** Интегрированная проверка вводимых данных на клиенте с помощью библиотеки `Yup`. Система динамически валидирует email, проверяет длину пароля и контролирует их совпадение при регистрации.
* **Локальная авторизация и сессии:** Имитация полноценного Auth-процесса с сохранением сессии пользователя, управлением ролями (`user` / `guest`) и генерацией уникального `auth_token` в `localStorage`.
* **Полная изоляция данных:** Все финансовые транзакции строго привязаны к `username` текущей сессии. Несколько учётных записей на одном устройстве гарантированно имеют изолированную историю операций.
* **Управление транзакциями:** Удобный интерфейс добавления доходов и расходов с указанием даты, категории, суммы и автоматическим расчетом баланса за выбранные периоды.
* **Интерактивная годовая аналитика:** Информативная визуализация финансовой динамики по месяцам текущего года с помощью гистограммы на базе `Chart.js`.
* **Умный рейтинг трат:** Автоматическая группировка расходов за год и автоматическая сортировка категорий от самых затратных к меньшим для наглядного аудита бюджета.

## Технологический стек

* **Ядро:** React (v19), Vite
* **Язык программирования:** TypeScript (строгая типизация интерфейсов и событий)
* **Управление состоянием:** React Context API (глобальный стейт авторизации)
* **Валидация:** Yup
* **Стилизация:** SCSS Modules (изолированные стили компонентов, адаптивная верстка под мобильные устройства)
* **Визуализация данных:** Chart.js
