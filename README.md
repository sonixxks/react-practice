# CashGlow — Личный финансовый трекер

**CashGlow** — это веб-приложение (SPA) для эффективного контроля и визуального анализа личных финансов.

## Как запустить проект локально

### Разработка и проверка

1. Клонируйте репозиторий с проектом и перейдите в рабочую директорию:
   ```bash
   git clone <https://github.com/sonixxks/react-practice.git>
   cd react-practice
2. Установите все зависимости проекта:
   ```bash
   npm install
3. Запустите сборку проекта (сервер + фронтенд):
   ```bash
   npm start
4. Откройте сайт в браузере по адресу в терминале 

### Сборка и Production

1. Скомпилируйте оптимизированный production:
   ```bash
   npm run build

## Ключевой функционал

* **Надежная валидация форм:** Интегрированная проверка вводимых данных на клиенте с помощью библиотеки `Yup`. Система динамически валидирует email, проверяет длину пароля и контролирует их совпадение при регистрации.
* **Полноценная JWT-авторизация:** Реализован собственный Node.js сервер аутентификации. Сессия пользователя защищена с помощью генерации валидных JWT-токенов, которые сохраняются в `localStorage` фронтенда и проверяются сервером при каждом запросе.
* **Централизованное хранение и изоляция данных:** Все данные пользователей и финансовых транзакций сохраняются на стороне сервера в единую базу данных `src/Data/db.json`. Финансовые операции строго привязаны к уникальному `userId` из расшифрованного JWT-токена, что гарантирует 100% изоляцию данных разных пользователей.
* **Управление транзакциями:** Удобный интерфейс добавления доходов и расходов с указанием даты, категории, суммы и автоматическим расчетом баланса за выбранные периоды.
* **Интерактивная годовая аналитика:** Информативная визуализация финансовой динамики по месяцам текущего года с помощью гистограммы на базе `Chart.js`.
* **Умный рейтинг трат:** Автоматическая группировка расходов за год и автоматическая сортировка категорий от самых затратных к меньшим для наглядного аудита бюджета.

## Технологический стек

* **Ядро фронтенда:** React (v19), Vite
* **Бэкенд-эмулятор:** Node.js (нативный модуль `http` без сторонних веб-фреймворков, аутентификация через `jsonwebtoken`)
* **База данных:** Файловая JSON-БД (`db.json`)
* **Язык программирования:** TypeScript / JavaScript
* **Управление состоянием:** React Context API (глобальный стейт авторизации)
* **Кроссплатформенный запуск:** Concurrently
* **Валидация:** Yup
* **Стилизация:** SCSS Modules (изолированные стили компонентов, адаптивная верстка под мобильные устройства)
* **Визуализация данных:** Chart.js
